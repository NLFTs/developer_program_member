## 🎯 Quick Start

Get started with NLFTs in 3 simple steps:

### Step 1: Create an Account

```bash
# Visit our developer portal
https://nlfts.dev/login

# Sign up with GitHub OAuth
# Get your API credentials
```

### Step 2: Install SDK

::code-group

```bash [npm]
npm install @nlfts/sdk
```

```bash [yarn]
yarn add @nlfts/sdk
```

```bash [pnpm]
pnpm add @nlfts/sdk
```

::

### Step 3: Make Your First API Call

```typescript
import { NLFTsClient } from '@nlfts/sdk'

// Initialize client
const client = new NLFTsClient({
  apiKey: 'your-api-key',
  environment: 'production'
})

// Make API call
const response = await client.getData()
console.log(response)
```
