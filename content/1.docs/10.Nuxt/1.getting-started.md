# Memulai dengan Nuxt

Nuxt adalah kerangka kerja open source yang membuat pengembangan web menjadi sederhana dan kuat.

## Instalasi

Gunakan perintah berikut untuk membuat proyek baru:

```bash
npx nuxi@latest init <project-name>
```

## Langkah Berikutnya
Setelah instalasi selesai, Anda dapat mulai membangun komponen dan halaman pertama Anda.
