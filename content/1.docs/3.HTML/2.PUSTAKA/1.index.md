# Pustaka

This is an example page showing how additional navigation items appear. 
In Nuxt Content, the sidebar menu is automatically generated based on the **folder and file structure**.

## Where is the Navigation Name Stored?

1. **Folder Name**: The folder name `2.advanced` is automatically parsed. The number `2.` sets the sorting order, and `advanced` becomes the title (capitalized).
2. **File Frontmatter**: At the top of this markdown file, we use a YAML block (between `---`).
   - `title`: Sets the name on the page and sidebar.
   - `navigation.icon`: Sets the icon next to it in the sidebar.

Try creating more Markdown files inside this folder!
