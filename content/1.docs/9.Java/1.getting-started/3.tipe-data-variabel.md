# 🧱 Modul 2 — Tipe Data & Variabel

Java adalah bahasa yang **strongly typed**. Artinya, setiap variabel **wajib** dideklarasikan tipe datanya sebelum digunakan. Ini membantu mencegah kesalahan dan membuat kode lebih terstruktur.

::tip{icon="i-lucide-lightbulb"}
**Tujuan Modul:** Setelah mempelajari modul ini, Anda akan mampu membedakan tipe data primitif dan reference, mendeklarasikan variabel dengan benar, menggunakan konstanta, melakukan konversi tipe, dan menerapkan aturan penamaan yang baik.
::

---

## 📊 1. Tipe Data Primitif (8 Jenis)

Tipe data primitif adalah tipe data bawaan Java yang menyimpan nilai langsung. Ada 8 jenis:

::code-preview
| Tipe | Ukuran | Rentang Nilai | Contoh |
|------|--------|---------------|--------|
| `byte` | 8-bit | -128 s/d 127 | `byte umur = 25;` |
| `short` | 16-bit | -32.768 s/d 32.767 | `short tahun = 2025;` |
| `int` ⭐ | 32-bit | ±2,1 miliar | `int gaji = 5000000;` |
| `long` | 64-bit | ±9,2 × 10¹⁸ | `long populasi = 7800000000L;` |
| `float` | 32-bit | ~7 digit desimal | `float suhu = 36.5f;` |
| `double` ⭐ | 64-bit | ~15 digit desimal | `double pi = 3.14159265;` |
| `char` | 16-bit | 0 s/d 65.535 (Unicode) | `char huruf = 'A';` |
| `boolean` ⭐ | — | `true` atau `false` | `boolean aktif = true;` |
#code

::

::callout{icon="i-lucide-star" color="primary"}
**Fokus untuk Pemula:** Pelajari 4 tipe ini dulu — cukup untuk 90% kasus:
- `int` → angka bulat (umur, jumlah, nilai)
- `double` → angka desimal (harga, persentase, rata-rata)
- `boolean` → benar/salah (status login, kondisi aktif)
- `String` → teks (nama, alamat, pesan) — ini Reference, bukan primitif!
::

## 🧩 2. Tipe Data Reference

Tipe data reference menyimpan **alamat memori** di mana objek disimpan (di heap memory), bukan nilai langsung.

::card-group
  ::card
  ---
  title: "String"
  icon: i-lucide-text
  ---
  Tipe untuk teks. Diapit tanda kutip ganda: `"Halo Dunia"`. Sebenarnya adalah class, tapi bisa digunakan seperti primitif.
  ::
  ::card
  ---
  title: "Array"
  icon: i-lucide-table
  ---
  Kumpulan nilai bertipe sama dalam satu variabel. Contoh: `int[] angka = {1,2,3};`
  ::
  ::card
  ---
  title: "Class & Object"
  icon: i-lucide-box
  ---
  Dasar dari OOP. Kita akan pelajari nanti di modul terpisah.
  ::
::

## ✍️ 3. Deklarasi & Inisialisasi Variabel

Berbagai cara mendeklarasikan dan menginisialisasi variabel di Java.

::tabs
  ::tabs-item{label="Deklarasi + Inisialisasi" icon="i-lucide-pencil"}
  Cara paling umum:
  ```javascript
  // Format: tipe_data nama_variabel = nilai;
  int umur = 20;
  double ipk = 3.75;
  boolean lulus = true;
  String nama = "Budi Santoso";
  char grade = 'A';
  ```
  ::
  ::tabs-item{label="Deklarasi dulu, isi nanti" icon="i-lucide-clock"}
  ```javascript
  int nilai;
  nilai = 85;
  ```
  ::
  ::tabs-item{label="Banyak variabel sekaligus" icon="i-lucide-layers"}
  ```javascript
  int a = 1, b = 2, c = 3;
  ```
  ::
::

::warning{icon="i-lucide-alert-triangle"}
Variabel lokal (di dalam method) **harus diinisialisasi** sebelum digunakan. Jika tidak, kompilasi akan gagal.
::

## 🔒 4. Keyword `final` — Konstanta

Gunakan `final` agar nilai variabel **tidak bisa diubah** setelah diinisialisasi.

```javascript
final double PI = 3.14159;
final int MAX_MAHASISWA = 30;
final String NAMA_KAMPUS = "Universitas Indonesia";

// PI = 3.0;  ← ERROR! Nilai final tidak bisa diubah
```

::tip{icon="i-lucide-info"}
**Konvensi:** Nama konstanta ditulis `HURUF_BESAR_DENGAN_UNDERSCORE`.
::

## 🔄 5. Konversi Tipe Data (Type Casting)

### ✅ Widening (Otomatis)
Dari tipe kecil ke besar — aman, tidak ada data loss.

```javascript
int angka = 100;
double hasil = angka;  // int → double, otomatis OK
long besar  = angka;   // int → long, otomatis OK
```

### ⚠️ Narrowing (Manual)
Dari tipe besar ke kecil — bisa kehilangan data, harus dilakukan secara eksplisit.

```javascript
double nilai = 9.99;
int bulatkan = (int) nilai; // Hasilnya: 9 (dipotong, BUKAN dibulatkan!)
```

### 🔄 Konversi String ↔ Angka

```javascript
String strAngka = "42";
int    parsed   = Integer.parseInt(strAngka);    // → 42
double parsedD  = Double.parseDouble("3.14");    // → 3.14

int    x  = 100;
String s  = String.valueOf(x); // → "100"
String s2 = x + "";            // cara singkat, hasil sama
```

::callout{icon="i-lucide-flask"}
**Peringatan:** Saat melakukan parsing, pastikan string berisi angka yang valid, jika tidak akan muncul error `NumberFormatException`.
::

## 📏 6. Aturan Penamaan Variabel

Aturan yang **WAJIB** diikuti, dan konvensi yang disarankan.

::accordion
  :::accordion-item{icon="i-lucide-check-circle" label="Aturan Mutlak (dari bahasa)"}
  - Boleh: huruf, angka, `_`, `$`
  - Tidak boleh diawali angka
  - Tidak boleh menggunakan keyword Java (seperti `class`, `int`, `for`)
  - Tidak boleh ada spasi
  - Case-sensitive (`nilai` ≠ `Nilai`)
  :::
  :::accordion-item{icon="i-lucide-thumbs-up" label="Konvensi (disarankan)"}
  - Gunakan **camelCase** untuk variabel: `namaLengkap`, `totalHarga`
  - Gunakan **PascalCase** untuk class: `HelloWorld`, `Mahasiswa`
  - Gunakan **UPPER_SNAKE_CASE** untuk konstanta: `MAX_VALUE`, `PI`
  :::
::

::code-preview
| Aturan | Contoh Benar | Contoh Salah |
|--------|--------------|--------------|
| Boleh: huruf, angka, `_`, `$` | `nama1`, `_nilai`, `$harga` | — |
| Tidak boleh diawali angka | `nilaiUjian`, `nilai1` | `1nilai`, `123abc` |
| Tidak boleh pakai keyword Java | `kelas`, `tipe` | `class`, `int`, `for` |
| Tidak boleh ada spasi | `namaDepan`, `nama_depan` | `nama depan` |
| Case-sensitive | `nilai` ≠ `Nilai` ≠ `NILAI` | — |
| Konvensi: **camelCase** | `namaLengkap`, `totalHarga` | `NamaLengkap` (itu PascalCase) |
#code

## 🧪 Latihan Praktik

Coba buat program sederhana yang:

1. Mendeklarasikan variabel untuk nama, umur, tinggi badan (dalam meter), dan status mahasiswa.
2. Gunakan tipe yang tepat.
3. Cetak semua variabel tersebut dengan format rapi.

Contoh output:
```
Nama: Siti Aminah
Umur: 19 tahun
Tinggi: 1.65 m
Mahasiswa: true
```

::tip{icon="i-lucide-lightbulb"}
Gunakan `System.out.printf()` atau `System.out.println()` dengan penggabungan string.
